# React + TypeScript + Vite

A simple frontend app that fetches the first 50 Pokémon from the Pokémon API (https://pokeapi.co) and displays them in a responsive grid with images and types.

###Features

- Fetches the first 50 Pokémon from the PokéAPI
- Displays official Pokémon artwork
- Shows Pokémon ID, name and types
- Responsive layout built with Flexbox
- Component-based architecture (Header, PokemonList, PokemonCard)

### Prerequisites

Make sure you have **Node.js** installed (version 18 or higher recommended).

### Installation 

- git clone <repository-url>
- move into progect folder with cd pokemon-app
- install dependencies with npm install

### Run the progect
npm run dev and you project should open in http://localhost:5173
